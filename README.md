# Adnan Ashraf Resume Website

This project is a personal resume website built using simple frontend
technologies.\
It presents my professional profile, skills, projects, education, and
experience in a clean and responsive single page layout suitable for
both online viewing and PDF export.

------------------------------------------------------------------------

## About the Project

This resume website was created to showcase my profile as an aspiring
full stack developer with a strong frontend foundation.\
The goal was to design a clean professional resume layout that works
well for recruiters, portfolio sharing, and LinkedIn PDF uploads.

The project focuses on simplicity, readability, and professional
presentation rather than heavy design effects.

------------------------------------------------------------------------

## Features

-   Clean single page resume layout\
-   Responsive design for different screen sizes\
-   Optimized spacing for one page PDF export\
-   Professional typography and structure\
-   Projects section with live demo and GitHub links\
-   Education and experience aligned in corporate resume style\
-   Print friendly CSS for PDF conversion

------------------------------------------------------------------------

## Technologies Used

-   HTML5 for semantic structure\
-   CSS3 for layout, typography, and responsiveness\
-   Remix Icons for contact and project icons

------------------------------------------------------------------------

## How to Use

1.  Clone or download this repository\
2.  Open the HTML file in your browser\
3.  To export as PDF press Ctrl + P and choose Save as PDF\
4.  Adjust scale or margins if needed for single page output
